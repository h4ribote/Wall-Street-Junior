# Indices (指数)

## 1. 定義
Index（指数）は、特定の市場セクターやテーマに基づくポートフォリオを単一の資産としてパッケージ化した金融商品です。
Paper Street におけるIndexは、単なる数値指標ではなく、**それ自体が売買可能な資産 (Tradable Asset)** として機能します。


## 2. メカニズム (現物バスケットモデル)
Indexの価格と価値は、現実世界のETF（上場投資信託）と同様に、**現物バスケット (Physical Basket)** の概念に基づいて設計されています。

### 単純現物バスケット価格 (Simple Physical Basket Price)
Indexの価格は、その構成銘柄の現在価格の**単純合計**として定義されます。
このモデルでは、Indexは**各構成銘柄を1株ずつ含むパッケージ**として定義されます。

*   **計算式**: `Index Price = Σ (Component Price_i)`
    *   Indexを1単位保有することは、その構成銘柄を**すべて1株ずつ保有することと等価**です。
    *   従来の時価総額加重平均や複雑なウェイト計算を排除し、極めて直感的で透明性の高い価格形成を実現します。

### Creation / Redemption (設定・交換)
Indexが市場価格と理論価格（構成銘柄の合計価値）から乖離しないよう、**指定参加者 (AP: Authorized Participants)** による裁定取引メカニズムが組み込まれています。
このプロセスには手数料が発生し、これが裁定取引のバンド（乖離許容幅）を形成します。

1.  **Creation (設定)**:
    *   APは、市場でIndexの構成銘柄（現物）を**各1株ずつ**買い集めます。
    *   集めた現物バスケットをシステム（Issuer）に拠出し、対価として新規発行された「Indexユニット」を**1単位**受け取ります。
    *   **手数料 (Creation Fee)**: 発行額の **0.10%** が徴収されます。
    *   これにより、Indexの供給が増え、現物の需要が増えます。

2.  **Redemption (交換)**:
    *   APは、市場で流通している「Indexユニット」を買い集めます。
    *   Indexユニットをシステムに返還（消却）し、対価として構成銘柄（現物バスケット）を**各1株ずつ**受け取ります。
    *   **手数料 (Redemption Fee)**: 交換額の **0.10%** が徴収されます。
    *   これにより、Indexの供給が減り、現物の供給が増えます。

### Arbitrage (裁定取引)
このメカニズムにより、Index価格は構成銘柄の合計価値（理論価格）の周辺に収束しますが、手数料コストの分だけ完全一致はしません。

*   **裁定バンド (Arbitrage Band)**:
    *   往復のコスト（Creation + Redemption または売買スプレッド）を考慮すると、理論価格から **±0.20% 程度** の乖離は「正常範囲」として許容され、裁定取引が発生しません。
    *   このバンドを超えて初めて、APによる裁定が利益を生むようになります。

*   **Index価格 > 理論価格 (Premium)**:
    *   APは現物を安く買い、Creationを行ってIndexを高く売ることで利益を得ます。
    *   -> Index売り圧力により価格是正。

*   **Index価格 < 理論価格 (Discount)**:
    *   APはIndexを安く買い、Redemptionを行って現物を高く売ることで利益を得ます。
    *   -> Index買い圧力により価格是正。


## 3. 特徴
*   **分散投資**: 1つのIndexを買うだけで、多数の銘柄に分散投資する効果が得られます。
*   **透明性**: 構成銘柄が公開されており、理論価格を単純な足し算で検証可能です。
*   **流動性**: 個別の構成銘柄よりも活発に取引される可能性があります。
